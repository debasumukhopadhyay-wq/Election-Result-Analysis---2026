# Specification Document

## West Bengal Assembly Election 2026 Prediction System

---

## 1. Overview

This document defines the technical and functional specification for a web-based application that predicts election results for all 294 Assembly seats in the West Bengal Assembly Election 2026.

The system will use a weighted algorithmic model powered by structured data inputs and Large Language Model reasoning (Anthropic Claude Sonnet / Opus 4.6) for qualitative analysis.

---

## 2. Objectives

* Predict constituency-level election outcomes
* Provide candidate-wise scoring and probability of winning
* Generate explainable reports
* Allow booth-level drill-down analysis
* Enable full state election simulation

---

## 3. Technology Stack

### Frontend

* React.js (with TypeScript)
* UI Framework: Material UI / Tailwind CSS
* State Management: Redux Toolkit / Zustand
* Charts: Recharts / D3.js

### Backend

* Node.js (Express.js)
* Data Processing Layer: Python microservices (optional)
* Database: PostgreSQL + Redis cache

### AI Layer

* Anthropic Claude Sonnet or Opus 4.6 (for reasoning and report generation)
* No direct Claude API integration required (pluggable design)

## 5. Prediction Algorithm

### 5.1 Weighted Scoring Model

Each candidate is assigned a score based on:

| Factor                     | Weight Range |
| -------------------------- | ------------ |
| Demography Match           | 15-25%       |
| Historical Results         | 20-30%       |
| Candidate Quality          | 15-20%       |
| Local Leadership Alignment | 10-15%       |
| Recent Events Impact       | 5-15%        |
| Party Popularity           | 10-20%       |
| Alliances                  | 5-10%        |

Final Score = Weighted Sum of All Factors

---

### 5.2 Booth-Level Simulation

* Each booth assigned micro-score
* Aggregation to constituency result
* Swing simulation based on turnout changes

---

### 5.3 AI Reasoning Layer

Claude Sonnet / Opus 4.6 used for:

* Interpreting news impact
* Candidate qualitative scoring
* Generating natural language reports
